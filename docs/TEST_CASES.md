# 测试用例

- 首页：标题、主操作、免责说明在 375px 可见，无横向滚动。
- 四问：一次一题；空值、单题 800 字、总计 3000 字校验；可前后导航。
- 恢复：刷新后恢复答案和当前问题；完成后恢复结果。
- 分析：按钮提交后禁用式切换到加载页；成功结果通过 Schema。
- 异常：超时、异常 JSON、缺少 API Key 均安全进入或提示演示路径，不白屏。
- 安全：浏览器无密钥/模型地址；日志无完整回答；重复点击不产生并发提交。
- 限流：IP 小时 10 次、项目每日 300 次后返回友好的演示结果。
- 结果：3 项资产、最多 2 个备选、最多 3 个且单项不超过 30 分钟的行动；至少 1 项接触真人。
- 积分：完整流程显示 100 分及“开局行动者”。
- 兼容：iOS/Android 常用浏览器及微信内置浏览器中文输入、键盘收起、返回、长卡片滚动、安全区。
- 网络：无 Google Fonts/CDN/远程脚本；首屏核心资源均由本站提供。
