# 演示说明

## 3 分钟演示顺序

1. **打开首页**（30秒）
   - 展示翡翠绿主视觉卡、品牌主张
   - 指出进度条 0%、经验值 0、Lv1 初见
   - 点击「进入第一关」

2. **聊天式四问**（60秒）
   - 展示聊天气泡界面、Value Mirror 提问
   - 快速输入珍妮案例回答（每题2-3句）
   - 展示第四问的轻量选项
   - 点击「完成照见，生成开局方案」

3. **动态追问演示**（30秒）
   - 加载页轮播文案
   - 演示模式返回动态追问
   - 回答追问后进入最终生成

4. **三张结果卡**（45秒）
   - 人生资产卡：3 项资产 + 价值定位
   - 最小产品卡：产品名、目标用户、建议测试价格
   - 24小时行动卡：3 项行动，至少 1 项真实用户验证
   - 「开局行动者」徽章 + 100 经验值

5. **品牌报告保存**（15秒）
   - 点击「保存我的开局报告」
   - 展示隐私确认弹窗
   - 生成品牌长图
   - 演示保存/分享/复制链接

## 珍妮案例

- 10 年以上互联网医疗运营经验
- 擅长医生资源拓展、项目运营和团队培训
- 擅长把复杂经验整理成 SOP
- 经常帮助别人梳理副业方向、知识库和 AI 实践
- 希望线上工作、兼顾家庭、低成本开始
- 希望把经验做成服务或工具

## 动态追问演示

演示模式第一次请求返回：
```
你提到经常帮别人梳理副业方向，最近一次是谁找你？她具体卡在哪里？
```

回答后，第二次请求返回完整演示结果。

## API 失效处理

- 无 API Key：自动进入演示模式，明确标注
- API 超时：30 秒后兜底为演示结果
- API 异常：返回演示结果 + 错误提示

## 网络异常处理

- fetch 失败：兜底为演示结果
- 2 秒后自动跳转结果页
- 显示「AI 分析暂时不可用，已为你展示演示结果」

## 手机现场操作

1. 用手机浏览器打开预览链接
2. 体验完整流程
3. 展示底部导航适配
4. 展示品牌长图长按保存
