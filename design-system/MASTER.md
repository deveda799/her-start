# Her Start Design System

## Direction

Warm, restrained and professional knowledge-product interface for experienced women exploring commercial value. Mobile-first at 375px; desktop content is centered. Avoid pink-led palettes, technology blue, gradients, glossy effects and decorative motion.

## Tokens

- Primary: `#1D5B49`; deep: `#113D32`; soft: `#DCEAE3`
- Canvas: `#F8F4EC`; surface: `#FFFDF8`; border: `#DEDBD0`
- Value accent: `#A77B2E`; soft gold: `#F2E7C9` (small areas only)
- Text: `#20332D`; secondary: `#66736D`
- Radius: 15px controls, 18-25px cards, 32px hero
- Spacing: 4/8px rhythm; generous 24/32/48px section gaps
- Type: local system Chinese sans stack; KaiTi only for short emotional statements

## Interaction

One primary action per screen. Minimum control height 52px, visible keyboard focus, inline form errors, semantic labels, no hover dependency. Motion is limited to state feedback and loading progress, with `prefers-reduced-motion` support.
