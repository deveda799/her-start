# 创建你的第一个项目

## 创建项目（3 步）

### 第 1 步：命名项目
1. 登录 https://codebanana.com/（支持谷歌、苹果、邮箱登录）
2. 在 Studio 页面点击「+」按钮，选择创建方式：
   - **Create Project**：从头新建
   - **Import GitHub Repository**：从 GitHub 仓库导入
3. 选择 Create Project 后，填写项目名称（必填）

### 第 2 步：选择模板
项目命名后，进入模板选择页面：

- **从头开始**：选择「Custom」（默认选项），空白项目自由发挥
- **按类目选择**：左侧提供分类导航，按行业或需求筛选模板
  - All（全部）
  - Research & Analysis（研究分析）
  - Documents（文档）
  - Websites & Apps（网站与应用）
  - Data & Dashboards（数据看板）
- **搜索模板**：找不到合适的？直接在搜索框输入你想做的事情，比如"周报"、"官网"

> 不确定选哪个也没关系，选「Custom」就好。项目创建之后随时可以调整方向，不用在这一步纠结太久。

### 第 3 步：点击 Create
模板选好后，点击右下角「Create」按钮，项目即创建成功。

现在你面前就是一个全新的项目空间——对话框、文件区、预览区都准备好了。在对话框里直接告诉 Agent 你想做什么就行，不需要任何特殊格式，像跟同事说话一样自然。

## 项目创建后

每个项目自动对应一个群聊，包含三种模式：

| 模式 | 说明 |
|------|------|
| **Team Agent** | 和 Agent 协作，让它帮你写代码、做项目。一次一人使用，群主控制权限 |
| **Private Agent** | 你的私人助手，可以独立提问、做规划，不会打扰 Team Agent 的工作 |
| **Discussion** | 团队成员之间的讨论区，自由沟通，不涉及 Agent |

> 第一次用？建议先在 Team Agent 里打个招呼，比如"帮我做一个公司介绍页面"或者"帮我分析一下这份 Excel"，看看 Agent 怎么回应。很多用户就是从这一句话开始，慢慢发现原来 AI 能帮的比想象中多得多。

## 邀请团队

项目群聊可通过右边栏「人+」图标，在 Invite tab 中添加组织内成员或邀请外部好友加入。一个人用也完全没问题，但如果有同事一起参与，协作的感觉会完全不一样。

## 修改项目名称

进入 Studio → 项目卡片右键或点击「···」菜单 → 选择「Rename」
