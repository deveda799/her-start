# 使用 Private Agent

## Private Agent 是什么

Private Agent 是项目群聊中的一种聊天模式，主要用于**提问和规划**。当 Team Agent 正在被其他同事使用时，你不用干等着——打开 Private Agent，该查的查、该问的问、该想的想，一点不耽误。

## 与 Team Agent 的区别

| 对比项 | Team Agent | Private Agent |
|--------|-----------|------------|
| 模式 | Coding 模式，Agent 直接写代码 | Ask 模式，提问和规划 |
| 使用限制 | 每次只能一个人使用 | 不限，大家可以同时用 |
| 权限控制 | 群主控制，可收回 | 无需授权 |
| 使用场景 | 让 Agent 执行开发任务 | 调研、验证、梳理思路 |

> 一个典型的使用场景：同事正在用 Team Agent 开发功能，你在 Private Agent 里提前梳理下一个需求的方案。等同事做完你接手 Team Agent 时，方案已经想清楚了，直接动手，效率翻倍。

## 核心能力

### 辅助理解
AI（Agent）和你共享同一个项目的完整上下文：
- 全局上下文同步：AI、开发、产品都基于同一个 repo、同一批文件结构、同一批知识
- 不需要手动贴文件，也不用反复解释"这个项目是做什么的"
- 新成员加入项目后，在 Private Agent 里问几个问题就能快速上手，比翻文档高效得多

## 权益扣费

- Private Agent 权益**也跟随项目 Owner 扣除**
- 因为 Private Agent 对话也被认为是为项目进行的工作

## 消息动作

- 支持 **Forward**（转发）：Private Agent tab 的转发**不受限制**（与 Team Agent 不同）
- 支持**引用、复制**消息
