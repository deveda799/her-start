# 连接 GitHub

## 如何导入 GitHub 仓库

四步完成：

1. 点击左侧边栏的 **「+」** 号
2. 选择 **「Import GitHub Repository」**
3. 点击 **「Authorize GitHub Account」**，登录你的 GitHub 账号完成授权
4. 授权成功后即可看到你的代码仓库列表，**选择一个仓库**，即可将项目导入到 CodeBanana

> 已有的项目不用从头开始。把 GitHub 仓库导进来，Agent 会自动读懂项目结构和代码逻辑——你可以直接让它在现有代码基础上做修改、加功能、修 Bug。

## 导入后的项目标识

项目列表会根据来源展示不同的前缀图标：

| 图标 | 来源 |
|------|------|
| GitHub 图标 | 从 GitHub 导入的项目 |
| 文件夹图标 | 本地创建的项目 |

## Git Status

位于 Studio 页面**左下角**，支持收起：

| 操作 | 说明 |
|------|------|
| 点击「保存」图标 | 提交新的 commit |
| 点击「历史」图标 | 查看全部 commit 记录，支持回退到之前的某个提交节点 |

> 每一次和 Agent 的对话都会自动生成 commit，所有变更都有迹可循。万一改出了问题，点一下历史记录就能回到之前任意一个状态，完全不用慌。

## Diff View

点击 Git Status 中有变更的文件，文件路径右侧会出现两个按钮：

| 按钮 | 作用 |
|------|------|
| **Preview** | 预览文件当前状态 |
| **Diff View** | 展示修改前后的对比 |

代码审查特别实用——Agent 改了哪些地方、改了多少行，一目了然。

## 实时协作同步

多人同时编辑同一个仓库，无冲突、无分支混乱：
- 即写即显
- 提交状态同步
- 变更实时广播给团队
