# Connect GitHub

## How to Import a GitHub Repository

Four steps:

1. Click the **"+"** button in the left sidebar
2. Select **"Import GitHub Repository"**
3. Click **"Authorize GitHub Account"** and log in to your GitHub account to complete authorization
4. Once authorized, you'll see your repository list — **select a repo** to import it into CodeBanana

> Existing projects don't need to start from scratch. Import your GitHub repo and the Agent will automatically understand the project structure and code logic — you can have it make changes, add features, or fix bugs directly on your existing codebase.

## Project Identification After Import

The project list displays different icons based on the source:

| Icon | Source |
|------|--------|
| GitHub icon | Imported from GitHub |
| Folder icon | Created locally |

## Git Status

Located in the **bottom-left** of the Studio page, collapsible:

| Action | Description |
|--------|-------------|
| Click the "Save" icon | Commit a new change |
| Click the "History" icon | View all commit history; supports rolling back to a previous commit |

> Every conversation with the Agent automatically generates a commit — all changes leave a trail. If something breaks, just click the history and roll back to any previous state. No need to panic.

## Diff View

Click a file with changes in Git Status, and two buttons will appear to the right of the file path:

| Button | Function |
|--------|----------|
| **Preview** | Preview the current state of the file |
| **Diff View** | Show a before-and-after comparison of changes |

Extremely useful for code review — see exactly what the Agent changed and how many lines were modified, at a glance.

## Real-Time Collaboration Sync

- Changes appear instantly
- Commit status syncs across the team
- Every change is broadcast in real time
