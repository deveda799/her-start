# Create Your First Project

## Create a Project (3 Steps)

### Step 1: Name Your Project
1. Log in at https://codebanana.com/ (supports Google, Apple, and email login)
2. On the Studio page, click the "+" button and choose a creation method:
   - **Create Project**: Start from scratch
   - **Import GitHub Repository**: Import from an existing GitHub repo
3. After selecting Create Project, enter a project name (required)

### Step 2: Choose a Template
After naming, you'll enter the template selection page:

- **Start from scratch**: Select "Custom" (default option) — a blank project with full flexibility
- **Browse by category**: Use the left sidebar to filter templates by industry or use case
  - All
  - Research & Analysis
  - Documents
  - Websites & Apps
  - Data & Dashboards
- **Search templates**: Can't find what you need? Type what you want to build in the search box, like "weekly report" or "landing page"

> Not sure which to pick? Just go with "Custom." You can always change direction after the project is created — no need to overthink this step.

### Step 3: Click Create
Once you've chosen a template, click the "Create" button in the bottom-right corner. Your project is now live.

You'll see a fresh project workspace — chat box, file area, and preview panel are all ready. Just tell the Agent what you want to do in the chat box. No special format needed; talk to it like you would a coworker.

## After Creating a Project

Each project automatically comes with a group chat that includes three modes:

| Mode | Description |
|------|-------------|
| **Team Agent** | Collaborate with the Agent — it writes code and builds your project. One person at a time; the Owner controls access |
| **Private Agent** | Your personal assistant — ask questions, plan ahead, without interrupting Team Agent's work |
| **Discussion** | An open discussion space for team members — free conversation, no Agent involved |

> First time here? Try saying something in Team Agent, like "help me build a company intro page" or "analyze this Excel file for me." Many users start with just one sentence and quickly discover that AI can do way more than they expected.

## Invite Your Team

In the project chat, click the "person+" icon in the right sidebar. Under the Invite tab, add organization members or invite external friends. Working solo is totally fine, but having teammates in the same space makes collaboration feel completely different.

## Rename a Project

Go to Studio → right-click the project card or click the "···" menu → select "Rename"
