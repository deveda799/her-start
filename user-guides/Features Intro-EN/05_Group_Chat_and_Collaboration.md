# Group Chat & Collaboration

## Project-Chat Relationship

- Every project corresponds to one group chat
- Every group chat includes three chat modes: Team Agent, Private Agent, Discussion

## Three Chat Modes

| Mode | Purpose |
|------|---------|
| **Team Agent** | Team collaborates with AI to produce code and content |
| **Private Agent** | Personal AI assistant for independent questions and validation |
| **Discussion** | Open discussion space for team members |

## Collaboration Workflow

The three modes work together to form a complete team collaboration loop:

1. **Team Agent produces content** → All group members can see the Team Agent's conversation and output
2. **Discussion for feedback** → Members discuss, share opinions, and suggest changes in Discussion
3. **Private Agent for independent validation** → Members can verify, test, or ask deeper questions in Private Agent without affecting Team Agent's work

> In short: **Agent works → everyone discusses → each person validates independently.** Three modes, each with its own role. No need for extra meetings to sync progress — everyone is already in the same information stream.

## Discussion in Detail
- An **open discussion space** within the project — all members can participate
- Chat freely about the project
- Supports **@mentioning anyone**, including **@Agent** (yes, you can summon AI directly in the discussion)
- Hovering over a message reveals three action buttons (left to right):

| Button | Icon | Function |
|--------|------|----------|
| **Reply** | ↩ curved arrow | Reply to the message, creating a quote reference |
| **Copy** | ⧉ overlapping rectangles | Copy the message content |
| **Forward** | ↗ outward arrow | Forward the message to another person or group |

## Inviting Members

Click the **"person+" icon** in the project's right sidebar to open the member management panel:
- **Members** tab: View current members
- **Invite** tab: Invite organization members or external friends to join the project
  - External friends **join only the current project, not the organization**
  - To join an organization, users need to click "Join Organization" in Contacts (top-left corner)
- **Add Agent** tab: Add a previously created AI Agent to participate in collaboration

## Chat (Human-Only)

Click the **Chat** tab next to Projects in the left sidebar to enter the human chat area:
- Create a "New Chat" — supports 1v1 private chats and group chats
- Members joining or leaving will trigger **broadcast notifications**
- Message actions support **copy, quote, and forward**
- Supports searching organization Contacts and chat history to start conversations

## Unread Messages & @Mentions

- Chat, Notification, and Projects all display unread message and @mention status
- **@mentions are prioritized over unread messages** — when someone @'s you, it won't get buried

## Notifications

- Displays all system notifications, Agent permission alerts, organization join requests, friend requests, billing notifications, etc.
- Includes "View in Project" and "Upgrade" shortcuts for quick navigation to the relevant project
